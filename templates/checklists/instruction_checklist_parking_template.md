# Чек-лист оформления парковки

## Область применения

Для сотрудников, которым по локальным правилам разрешена парковка.

## Ответственные

| Роль | Зона ответственности |
|------|----------------------|
| Руководитель | подтверждение необходимости |
| АХО | оформление парковочного доступа |
| Сотрудник | предоставление данных по правилам работодателя |

## Простая проверка

- [ ] Проверено, что парковка действительно нужна для роли.
- [ ] Указан автомобиль или плейсхолдер: `[автомобиль]`.
- [ ] Указано парковочное место или плейсхолдер: `[место]`.
- [ ] Запись добавлена в parking list после согласования.

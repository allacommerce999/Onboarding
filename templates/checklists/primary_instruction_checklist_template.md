# Чеклист подготовки инструктажей при приеме

## Карточка

| Поле | Значение |
|------|----------|
| Сотрудник | `[ФИО]` |
| Роль | `[роль]` |
| Формат работы | `[офис/удаленно/гибрид/выезд]` |
| Дата выхода | `[YYYY-MM-DD]` |
| Ответственный HR | `[роль/плейсхолдер]` |

## Минимальный набор

- [ ] Определить формат работы и объект: офис, удаленно, гибрид или выезд.
- [ ] Выбрать обязательные инструктажи по матрице.
- [ ] Отметить процедуры, которые требуют профильного согласования.
- [ ] Подготовить формы фиксации: журнал, лист ознакомления, заявка, акт.
- [ ] Передать пакет на проверку HR/ОТ/ИБ.

## Критерий готовности

Пакет готов к выдаче сотруднику, если выбранные чек-листы заполнены, все неизвестные данные помечены как `требует уточнения`, а профильные допуски не отмечены как выданные автоматически.

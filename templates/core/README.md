# Core Templates

Базовые файлы для подготовки onboarding-пакета.

Полный сценарий запуска находится в [`../../QUICKSTART.md`](../../QUICKSTART.md).

| Файл | Назначение |
|------|------------|
| `employee_input_form.md` | форма входных данных сотрудника |
| `onboarding_package_template.md` | структура итогового onboarding-пакета |
| `quality_checklist.md` | проверка полноты результата |

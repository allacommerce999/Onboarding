# Access Templates

Матрицы доступов и шаблоны заявок на ИТ-ресурсы.

Полный сценарий запуска находится в [`../../QUICKSTART.md`](../../QUICKSTART.md).

| Файл | Назначение |
|------|------------|
| `workgroup_access_matrix.md` | матрица доступов рабочей группы |
| `access_matrix_template.md` | общий шаблон матрицы доступов |
| `access_request_template.md` | шаблон одной заявки |
| `access_request_bundle_template.md` | шаблон пакета заявок |
| `domain_accounts_register_template.md` | шаблон реестра доменных учетных записей |

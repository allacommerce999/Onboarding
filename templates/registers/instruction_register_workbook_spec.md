# Спецификация Excel-книги учета инструктажей

Файл: `instruction_register_template.xlsx`

> Synthetic workbook template. Книга создается заново и содержит только synthetic строки.

## Листы

| Лист | Назначение |
|------|------------|
| `README` | назначение, предупреждения, правила synthetic-данных |
| `instruction_log` | журнал записей инструктажей |
| `instruction_catalog` | каталог инструкций и программ |
| `role_matrix` | связь ролей с обязательными инструктажами |
| `examples` | 3-5 synthetic строк для проверки структуры |

## Колонки `instruction_log`

| Колонка | Пример |
|---------|--------|
| N | `1` |
| Дата инструктажа | `2026-05-12` |
| ФИО сотрудника | `Примеров Петр Петрович` |
| Табельный номер | `TEST-0001` |
| Подразделение | `TEST-DEPARTMENT` |
| Должность | `инженер тестовой команды` |
| Вид инструктажа | `вводный` |
| Название/код инструкции | `OT-INTRO-TEST` |
| Основание/программа | `TEST-PROGRAM` |
| Инструктирующий | `Инструктор Тестовый` |
| Должность инструктирующего | `специалист ОТ` |
| Результат проверки знаний | `пройдено` |
| Подпись инструктируемого | `synthetic-signature` |
| Подпись инструктирующего | `synthetic-signature` |
| Следующий инструктаж | `2026-11-12` |
| Примечание | `пример` |

## Требования к synthetic-данным

- Не вставлять реальные ФИО, подписи, табельные номера, кабинеты, номера авто, пропусков и телефонов.
- Создавать файл заново по этой спецификации и использовать только тестовые строки.
- Не добавлять внешние ссылки и подключения к данным.

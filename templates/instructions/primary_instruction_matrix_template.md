# Матрица первичных инструктажей и локальных процедур

> Template. Все значения заменить на данные работодателя после проверки HR, ОТ, ИБ и ответственных за объект.

## Карточка роли

| Поле | Значение |
|------|----------|
| Роль сотрудника | `[роль]` |
| Формат работы | `[офис/удаленно/гибрид/выезд]` |
| Объект | `[офис / объект / удаленно / TEST-SITE]` |
| Дата выхода | `[YYYY-MM-DD]` |
| Ответственный за пакет | `[роль ответственного]` |

## Матрица

| Категория | Для кого | Ответственный | Результат | Статус |
|-----------|----------|---------------|-----------|--------|
| ПВТР | все сотрудники | HR | лист ознакомления | `[нужно / готово]` |
| Охрана труда: вводный инструктаж | все сотрудники | специалист ОТ | запись в журнале | `[нужно / готово]` |
| Охрана труда: первичный на рабочем месте | рабочее место или объект | руководитель работ | запись в журнале | `[нужно / не требуется]` |
| Пожарная безопасность | все сотрудники | ответственный за ПБ | запись/лист ознакомления | `[нужно / готово]` |
| Первая помощь | по рискам роли | специалист ОТ | чеклист ознакомления | `[нужно / не требуется]` |
| ЛВС и ИБ | сотрудники с ИТ-доступом | ИБ / IT | лист ознакомления | `[нужно / готово]` |
| Оборудование | получающие имущество | IT Workplace / АХО | акт передачи | `[нужно / не требуется]` |
| Кабинет и ключи | офисные сотрудники | АХО | запись в списке доступа | `[нужно / не требуется]` |
| Парковка | по заявке | АХО | запись в parking list | `[нужно / не требуется]` |
| Электронный пропуск | доступ в здание | HR / АХО | заявка на пропуск | `[нужно / готово]` |
| Электроустановки | выездные роли | профильный руководитель | требует профильной проверки | `требует согласования` |
| Работа на высоте | роли с высотными работами | профильный руководитель | требует профильной проверки | `требует согласования` |

## Правила применения

- Агент подбирает строки по роли и формату работы.
- Агент не выдает допуски и не подтверждает проверку знаний.
- Все профильные работы помечаются как `требует согласования`.
- Реальные инструкции, журналы и программы обучения проверяются по действующей редакции локальных документов.

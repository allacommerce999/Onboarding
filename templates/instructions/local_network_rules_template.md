# Правила работы в ЛВС и ИБ

## Основные правила

- Использовать только выданную учетную запись и корпоративное устройство.
- Не передавать пароли, токены, коды MFA и приватные ключи.
- Блокировать рабочее место при отходе.
- Не подключать неразрешенные носители и сервисы.
- Сообщать об инцидентах по каналу `[security-contact-placeholder]`.

## Доступы повышенной чувствительности

| Доступ | Правило |
|--------|---------|
| Production | требует согласования |
| DBA/Admin | требует согласования |
| Персональные данные | требует согласования ИБ |
| Сервисная учетная запись | требует владельца и срока пересмотра |

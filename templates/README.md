# Onboarding Templates

Эта папка содержит шаблоны для подготовки onboarding-пакета: форму входных данных, матрицы доступов, заготовки заявок, чек-листы, интерактивные HTML-файлы и книгу учета инструктажей.

Полный сценарий запуска находится в [`../QUICKSTART.md`](../QUICKSTART.md).

## Структура

| Папка | Что внутри |
|-------|------------|
| `core/` | базовая форма входных данных, шаблон onboarding-пакета и quality checklist |
| `access/` | матрицы доступов и шаблоны заявок на ИТ-ресурсы |
| `instructions/` | шаблоны инструктажей и матрица обязательных процедур |
| `checklists/` | чек-листы по ПВТР, ОТ, ПБ, ИБ, пропускам, парковке, высоте и электроустановкам |
| `knowledge-checks/` | самодостаточные HTML-мини-проверки знаний после инструктажа |
| `local-procedures/` | шаблоны по оборудованию, пропускам, ключам и парковке |
| `registers/` | спецификация и Excel-книга учета инструктажей |
| `interactive/` | интерактивный HTML-чек-лист выбора и отметки процедур |

## Что открыть первым

1. `core/employee_input_form.md` — заполнить данные сотрудника.
2. `access/workgroup_access_matrix.md` — проверить доступы и владельцев систем.
3. `core/quality_checklist.md` — проверить готовый результат.

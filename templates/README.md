# Onboarding templates

Эта папка содержит шаблоны onboarding-агента: форму входных данных, заготовки пакетов, матрицы доступов, чек-листы, интерактивные HTML-файлы и книгу учета инструктажей.

## Структура

| Папка | Что внутри |
|-------|------------|
| `core/` | базовая форма входных данных, шаблон onboarding-пакета, prompt template и quality checklist |
| `access/` | матрицы доступов и шаблоны заявок на ИТ-ресурсы |
| `instructions/` | шаблоны инструктажей и матрица обязательных процедур |
| `checklists/` | простые чек-листы по ПВТР, ОТ, ПБ, ИБ, пропускам, парковке, высоте и электроустановкам |
| `knowledge-checks/` | самодостаточные HTML-мини-проверки знаний после инструктажа |
| `local-procedures/` | шаблоны по оборудованию, пропускам, ключам и парковке |
| `registers/` | спецификация и Excel-книга учета инструктажей |
| `interactive/` | интерактивный HTML-чек-лист выбора и отметки процедур |

## Быстрый старт

1. Если вы открыли только эту папку, сначала скачайте весь репозиторий `Onboarding`.
2. Заполните `core/employee_input_form.md`.
3. Откройте чат с Onboarding Agent.
4. Вставьте в чат текст ниже. После блока `Форма` вставьте заполненную форму, после блока `Матрица доступов` вставьте содержимое `access/workgroup_access_matrix.md`.

```text
Подготовь onboarding-пакет в режиме draft.

Используй заполненную форму сотрудника и матрицу доступов ниже.
Если данных не хватает, отметь их как `требует уточнения` и вынеси вопросы в раздел `Открытые вопросы`.
Чувствительные доступы и профильные допуски не отмечай как выданные автоматически.

Форма:
[вставьте сюда заполненный файл templates/core/employee_input_form.md]

Матрица доступов:
[вставьте сюда файл templates/access/workgroup_access_matrix.md]
```

Для финальной версии замените в первой строке `draft` на `final`. После получения результата проверьте пакет по `core/quality_checklist.md`.

## Рабочие правила

- Адаптировать владельцев, ресурсы и маршруты согласования под свою организацию.
- Профильные работы, электроустановки и высоту помечать как `требует профильной проверки`.
- HTML-файлы запускаются локально: откройте нужный `.html` двойным кликом, через пункт браузера "Открыть файл" или перетащите файл в окно браузера. Устанавливать зависимости не нужно.
- Excel-книгу использовать как шаблон учета инструктажей и при необходимости пересобирать под локальный формат.

# Templates

Актуальная публичная структура шаблонов:

- `documents/` - очищенные `.docx` и `.xlsx` формы;
- `core/` - markdown-резерв для входных данных и проверки качества;
- `knowledge-checks/` - HTML-проверки знаний.

Старые markdown-шаблоны перенесены в `_legacy/2026-05-markdown-agent/` и не используются как актуальный источник генератора.

# Documents

Папка содержит очищенные публичные шаблоны и карту соответствия внутренним источникам. Шаблоны используют тестовые названия ресурсов и не повторяют внутренние формы дословно.

Файлы `.docx` и `.xlsx` создаются генератором из безопасных блоков. Они нужны как публичные примеры формата, а не как юридически утвержденные корпоративные формы.

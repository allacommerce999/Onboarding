# Пример заявки на электронный пропуск

## Заявка

| Поле | Значение |
|------|----------|
| Тип заявки | первичный пропуск |
| Сотрудник | Примеров Петр Петрович |
| Роль | инженер TEST-APP |
| Подразделение | TEST-DEPARTMENT |
| Объект | TEST-SITE |
| Зона доступа | TEST-ZONE-A |
| Срок действия | 2026-05-18 - 2026-12-31 |
| Основание | onboarding |

## Согласование

| Роль | Статус |
|------|--------|
| Руководитель | согласовано |
| HR | согласовано |
| АХО / служба пропусков | требуется оформление |

## Комментарий

Агент только готовит заявку. Решение о выдаче пропуска принимает ответственная служба.

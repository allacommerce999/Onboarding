# Onboarding Examples

Эта папка содержит примеры документов, которые можно использовать как ориентир при работе с Onboarding Agent.

Полный сценарий запуска находится в [`../QUICKSTART.md`](../QUICKSTART.md).

## Структура

| Папка | Состав |
|-------|--------|
| `access/` | заявки на доступ, сопроводительное письмо, реестр доменных учетных записей |
| `onboarding-packages/` | полные onboarding-пакеты |
| `instruction-packages/` | пакеты первичных инструктажей для офисной и выездной роли |
| `instructions/` | краткие примеры инструкций по ПВТР, ОТ, ПБ, первой помощи, ЛВС/ИБ и выездным работам |
| `local-procedures/` | парковка, ключи, пропуск, имущество |
| `demos/` | сценарии проверки интерактивных HTML-артефактов |

Персональные onboarding-пакеты по сотрудникам лежат в [`../employees/`](../employees/).

## Как использовать

- Используйте файлы из `../templates/` как базовые формы.
- Используйте примеры из тематических подпапок как few-shot материалы для агента.
- Адаптируйте роли, владельцев, сроки, ресурсы и правила согласования под свою организацию.

# Пример пакета первичных инструктажей: офисный ИТ-сотрудник

## Карточка

| Поле | Значение |
|------|----------|
| Сотрудник | Примеров Петр Петрович |
| Роль | инженер корпоративных систем |
| Команда | TEST-APP-TEAM |
| Формат работы | офис |
| Объект | TEST-SITE |
| Дата выхода | 2026-05-18 |

## Обязательные процедуры

| Процедура | Ответственный | Артефакт | Статус |
|-----------|---------------|----------|--------|
| ПВТР | HR | лист ознакомления | нужно сделать |
| Охрана труда | специалист ОТ | запись в журнале | нужно сделать |
| Пожарная безопасность | ответственный за ПБ | лист/журнал | нужно сделать |
| ЛВС и ИБ | ИБ / IT | лист ознакомления | нужно сделать |
| Оборудование | IT Workplace | акт передачи | нужно сделать |
| Электронный пропуск | HR / АХО | заявка | требует согласования |
| Кабинет и ключи | АХО | список доступа | требует согласования |
| Парковка | АХО | parking list | не требуется |

## Минимальная проверка знаний

- Сотрудник знает рабочий режим и порядок отсутствий.
- Сотрудник знает, куда сообщать об инциденте ОТ/ПБ/ИБ.
- Сотрудник подтвердил получение оборудования.

## Открытые вопросы

- Нужна ли парковка.
- Нужен ли доступ к дополнительным помещениям.
- Кто фиксирует итог в локальной системе учета.
